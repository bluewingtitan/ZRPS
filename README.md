# Z Role Playing System

I wrote this system for a homebrew system of mine playing in the fictional world of "Zamonien" by Walter Moers. No relevant assets or IP of Zamonien is included in here for legal reasons, only a system for FoundryVTT.

I built this to fit my specific setup and homebrew system. So while I am happy for you to use it (maybe just as an example to see how typescript-driven development of Foundry-Systems could look like), but please understand that I won't offer support. The repo is mostly public because it makes things easier and I don't really see a reason why it shouldn't be.

Also understand that contrary to most of my projects, this one is making use of generative AI to implement a lot of the templates and data classes, following my game system instructions.
